About

Bringing desktop functionality to the web.

Simple

Root Tree works in real time, and our easy to install software along with a simple javascript SDK, makes it easy to start using it in your web app!

Powerful

Root Tree lets you run set up code, automatically sync files, and make everyday web use that much simpler on a users computer. More functions can also be requested by developers!

Secure

User always has the control of their permissions, and only authenticated requests can be executed. Users can depend which features they want to activate or deactivate.

How it works

  • User installs Root Tree

    Our software allows access to the user's computer, depending on the level of permissions given by the user.

  • Developer makes an API call

    Using our API (we also have a js SDK to get started quickly), web developers can create a request to excute a function on a user's computer. These functions are pre-approved by Root Tree and range from desktop notifications, file syncing to general installation procedures. Developers can request for additional functions if needed.

  • User authenticates the developer's app

    Upon making the request, Root Tree sdk asks the user to authenticate the developer to allow execution of the asked functions

  • Root Tree client executes

    Root Tree software on the user's computer will execute the commands asked by the developer and return the necessary results back.

  • Web just got more interactive!
